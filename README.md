# blog.joelibaceta.com

My personal blog. Yes, built with Jekyll. Yes, in 2025. No, I don't want to talk about it.

For the three people who still remember that Ruby exists outside of Rails — you're welcome, this one's for you.

It's just a static site. No Docker Compose with 14 services. No GraphQL layer. No hydration. Just markdown files becoming HTML like nature intended.

---

Built with [Jekyll](https://jekyllrb.com) + [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy). Hosted on GitHub Pages because sometimes boring is good.
